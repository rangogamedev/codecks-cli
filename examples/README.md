# Examples

Examples for teams that want to use `codecks-cli` with AI agents.

## Included

- `skills/setup/SKILL.md` - secure setup workflow for first-time installs
- `skills/pm/SKILL.md` - CLI-first PM session skill
- `game-dev-agent.md` - example of a game-development flavored PM agent prompt

## Recommended Order

1. Read [docs/ai-agent-guide.md](../docs/ai-agent-guide.md)
2. Run `codecks-cli setup`
3. Start with `codecks-cli agent-init --agent`
4. Copy the skill that matches your editor or workflow
